# 更新日志

本项目遵循语义化版本。所有显著变更都会记录在这里。

## [0.1.1] - 2026-07-17

### 修复

- 为无边框快速记录窗口补充 Tauri 原生拖拽权限，解决只能缩放、无法移动的问题。
- 将快速记录窗口整条顶部栏设为拖拽区域，同时保留固定、展开和关闭按钮的点击交互。

### 优化

- 重构浅色与深色 Design Tokens，提升背景、表面、边框、文字和聚焦状态的层级对比。
- 设置页改为可即时预览的浅色、深色、跟随系统主题卡片。
- 优化主界面笔记卡片、编辑器、快速记录窗口和滚动条的视觉与交互状态。

## [0.1.0] - 2026-07-14

### 新增

- Tauri 2、React、TypeScript、Vite 和 Tailwind CSS 项目骨架。
- 本地 SQLite 数据库、迁移、FTS5 中文全文搜索和兼容降级搜索。
- 笔记、分类、标签、待办、收藏、归档、软删除与恢复。
- 500ms 自动保存、乐观锁冲突处理、Markdown 编辑与预览。
- 快速记录窗口、全局快捷键、托盘、开机启动和单实例运行。
- 规则化自动整理、JSON/Markdown 导出、JSON 导入与数据库备份恢复。
- 自动整理实体与建议持久化，采用/忽略状态可跨会话保留。
- 隐藏、切换与退出前统一刷新草稿，保存失败时阻止退出并提示风险。
- 搜索栏支持标签、日期范围及全部待办状态筛选。
- Windows MSI、NSIS、独立 exe 的 GitHub Actions 构建配置。
- 按日期锁定 Rust stable 工具链、Windows Release 标签校验和 SHA-256 清单。
- 快速记录窗口支持拖动与固定；固定后保留位置且不因失焦隐藏。
- 快速记录隐藏分类与日期，主编辑器默认收起元数据，让正文成为主要区域。
- Windows 安装包内置离线 WebView2，NSIS 安装后创建桌面和开始菜单入口。
- CI 生成可直接分发的完整 `Release` 目录、统一命名产物和校验文件。

### 已知限制

- Windows 安装包暂未进行代码签名。
- 提醒、日历、云同步、付费 AI 和自动更新尚未实现。
